# Utilisation

## Raccourcis
- ALT : déplacer l’icône
- SHIFT + clic sur l’icône : options
- /Alt icon : afficher/masquer l’icône

## Fonctions principales (UI)
- Affiche la liste complete des emote disponible du jeu

## Données & synchro
- Les données sont enregistrées à chaque connexion/déconnexion.
